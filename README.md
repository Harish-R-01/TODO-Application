# WorkSync – Collaborative Task Management System

An end-to-end task management web application designed with role-based access control and secure authentication to ensure appropriate data visibility across user roles.

## Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, JavaScript

## Key Features
* **Role-Based Access Control:** Configured secure user authentication and permissions to manage role-specific data visibility.
* **Task Management:** Responsive user interface integrated with a relational SQLite database for efficient task storage, tracking, and retrieval.
* **Version Control:** Maintained structured development practices and code quality using Git.
