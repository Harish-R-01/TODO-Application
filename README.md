# TODO Application

A full-stack task management web application developed using Django to help users organize, track, and manage their daily tasks efficiently.

## Tech Stack
* **Backend Framework:** Python (Django)
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, JavaScript

## Core Features
* **Task Management (CRUD):** Users can seamlessly create, view, update, and delete tasks.
* **Status Tracking:** Mark tasks as completed or pending with dynamic data updates.
* **Relational Storage:** Integrated with a structured SQLite database to ensure clean data persistence.
