# TODO Application

A responsive web application designed to help users efficiently manage, track, and organize their daily tasks and priorities.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend/Database:** Java, SQLite

## Key Features
* **Task Management:** Create, read, update, and delete (CRUD) operations for daily tasks.
* **Status Tracking:** Mark tasks as completed, pending, or in-progress.
* **Responsive UI:** Clean and intuitive user interface designed for both desktop and mobile viewing.
